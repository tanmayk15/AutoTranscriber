# Screenshots Guide

This directory contains screenshots for the README documentation.

## Required Screenshots

To complete the documentation, please add the following screenshots:

1. **main-interface.png**
   - Full application interface showing upload area
   - Video preview section
   - Processing buttons visible

2. **subtitle-generation.png**
   - Subtitle generation in progress
   - Progress bar showing transcription
   - Generated subtitles visible in viewer

3. **translation.png**
   - Translation panel with language selection
   - Original and translated subtitles side-by-side
   - Download buttons visible

4. **burned-video.png**
   - Video player showing burned subtitles
   - Final output video with subtitles embedded
   - Quality comparison

## How to Take Screenshots

1. Start the development server: `npm run dev`
2. Upload a test video
3. Process through each step
4. Take screenshots at each stage
5. Save with the exact filenames above
6. Optimize images (compress to reduce file size)

## Screenshot Specifications

- **Format:** PNG (lossless quality)
- **Resolution:** 1920x1080 or higher
- **File Size:** < 500KB each (use compression)
- **Content:** Clean interface, no personal information

## Tools for Screenshots

- **Windows:** Snipping Tool, Win + Shift + S
- **Mac:** Command + Shift + 4
- **Linux:** Screenshot tool, Flameshot

## Image Optimization

Use these tools to compress images:
- TinyPNG (https://tinypng.com/)
- Squoosh (https://squoosh.app/)
- ImageOptim (Mac)
